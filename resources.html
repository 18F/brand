Download theme

How to use:

  1. Install Droid Sans and Droid Serif 
  2. Download 18F_Keynote_Theme
  3. Open in keynote
  4. Add to “My Themes”
  5. Create a new document using the theme chooser
  6. Add pages from the Master Pages palette
  7. Enjoy!
