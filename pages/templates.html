Google Docs templates

We have two Google Docs templates: a basic outline and a detailed report example with a table of contents, tables, typographic elements, and an appendix. While both of these templates are branded for 18F, anyone in TTS can use them.

Instructions

To save 18F styles as your default Google styles:

  1. Open the basic outline.
  2. In the Styles drop-down menu, choose Options > Save as my default styles.
  3. Click File > New > Document or open an existing document to apply the 18F styles to it.
  4. In the Styles drop-down menu, choose Options > Use my default styles.

To reuse specific tables or typographic elements in a Google Doc:

  1. Open the detailed report example.
  2. Click File > Make a copy… (This will ensure you don’t accidentally edit the original master file.)
  3. Highlight and delete any sections you don’t need, and make changes as needed.

Presentation templates for Google Slides and Keynote



Google Slides template Download Keynote template

Instructions

Google Slides

  1. In Google Drive, click New > Google Slides.
  2. Once a new presentation opens, choose the Theme button in the toolbar.
  3. Click Import theme at the bottom of the theme menu.
  4. Search for 18F_Theme_GoogleSlides.
  5. Select and import the theme.

You can choose different layouts by selecting and inserting master pages from the template. For more details, learn about custom layouts and themes.

Keynote

  1. Open the 18F_Template_2016-09-15.key file.
  2. In the File menu, click Save Theme… and click Add to Theme Chooser.

To start a new presentation with the template, press Command + N, click My Templates, and choose the 18F template. For more details, learn about changing the look and layout of slides.
