Instructions for Keynote

Install the color palette to use it in any iWork application:

  1. Open the 18F_Color_Palette.zip file to unzip it.
  2. In Keynote, press the Shift + Command + C keys to open the Colors menu.
  3. Click the Color Palettes tab.
  4. Click the gear icon > Openâ€¦
  5. In the 18F_Color_Palette folder, select 18F_brand_colors.clr and click Open.

Guidelines

Do use the 18F brand colors in combinations that conform with accessibility standards. Each of these combinations meets or exceeds a contrast ratio of 4.5:1, and is approved for body and headline text. For more details on color contrast, read the 18F Accessibility Guide.
